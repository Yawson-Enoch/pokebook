# [Pokebook](https://pokebook-5gy.pages.dev/)

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Yawson-Enoch/pokebook.git
   ```
2. Install packages
   ```sh
   npm install
   ```
3. Run dev server
   ```sh
   npm run dev
   ```
4. Preview site (build and preview)
   ```sh
   npm run build:preview
   ```
